# LabT Application

## Description
LabT est une application Streamlit pour le calcul et l'analyse de signaux, incluant :

- Calcul inconnu
- Calcul Signal-to-Noise (S/N) classique et USP
- LOD et LOQ basés sur la pente de la courbe de linéarité (optionnel)
- Export des résultats en PDF
- Interface bilingue Français / Anglais
- Gestion des utilisateurs (Admin / User)
