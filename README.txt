# LabT

## Description
Application Streamlit pour la linéarité et calcul S/N avec export PDF/CSV.

## Déploiement
1. Crée un nouveau projet sur [Streamlit Cloud](https://share.streamlit.io/)
2. Upload le dossier complet `LabT/`
3. Assure-toi que `requirements.txt` est pris en compte pour l'installation des packages
4. Lance l'app
